# Order Triage &amp; Management — Engagement Hub

Prepared by Microsoft Solution Engineering for **Moen · Fortune Brands**.

An interactive walkthrough of a Microsoft-native approach to order triage: emailed purchase
orders (body or PDF) are read by AI, validated against Dynamics 365 customers and products, and
turned into orders — with only true exceptions routed to a person.

**Open [`index.html`](./index.html)** (the engagement hub) for the recap video, the capability
overview, and the interactive prototype.
